# ✂️ BEN2 · Background Remover

Streamlit-застосунок для нейромережевого видалення фону із зображень на основі моделі [PramaLLC/BEN2](https://huggingface.co/PramaLLC/BEN2).

---

## 🚀 Можливості

- **Пакетна обробка** — завантажте одразу кілька зображень і обробіть їх одним кліком
- **Інтерактивний слайдер** Before / After для порівняння результату
- **Вибір формату виводу** — PNG або WebP із прозорим фоном
- **Метрики якості** — відсоток покриття об'єкта та чіткість країв
- **Завантаження результатів** — окремі файли або ZIP-архів одразу для всіх
- **Автовизначення пристрою** — автоматично використовує CUDA, якщо доступна

---

## 🛠 Встановлення

### 1. Клонуйте репозиторій

```bash
git clone https://github.com/svniko/bg-remover.git
cd bg-remover
```

### 2. Створіть віртуальне середовище

```bash
python -m venv .venv
source .venv/bin/activate      # Linux / macOS
.venv\Scripts\activate         # Windows
```

### 3. Встановіть залежності

```bash
pip install -r requirements.txt
```

### 4. Запустіть застосунок

```bash
streamlit run app.py
```

## ⚙️ Використання

1. Відкрийте застосунок у браузері (`http://localhost:8501`)
2. У бічній панелі оберіть формат збереження: **PNG** або **WebP**
3. Перетягніть або виберіть одне чи кілька зображень
4. Натисніть **✂️ Remove Backgrounds**
5. Переглядайте результати через слайдер Before / After
6. Завантажте окремий файл або весь пакет одним ZIP-архівом

---

## 📊 Метрики результату

Для кожного зображення застосунок показує:

- **Об'єкт (%)** — частка пікселів, які належать виділеному об'єкту (alpha > 128)
- **Чіткість країв** — середня інтенсивність крайових пікселів маски (через `ImageFilter.FIND_EDGES`), що відображає якість відокремлення об'єкта від фону

---

## 🧠 Модель

Застосунок використовує **BEN2 (Background Erase Network v2)** від [PramaLLC](https://huggingface.co/PramaLLC/BEN2) — модель для сегментації переднього плану із високою точністю країв.

Модель завантажується автоматично при першому запуску та кешується засобами Streamlit (`@st.cache_resource`).
